# Contributing to Orchestra Framework

## Getting Started
To contribute to Orchestra Framework, follow these steps:
1. Fork the repository: `git fork https://github.com/orchestra-framework/orchestra-framework.git`
2. Clone the fork: `git clone https://github.com/your-username/orchestra-framework.git`
3. Create a new branch: `git branch my-branch`
4. Make changes and commit: `git commit -m 'my changes'`
5. Push changes to fork: `git push origin my-branch`
6. Create pull request: `git request-pull my-branch`

## Code Style
Orchestra Framework follows the PEP 8 style guide for Python code.

## Testing
To test Orchestra Framework, follow these steps:
1. Install dependencies: `pip install -r requirements.txt`
2. Run tests: `python -m unittest discover -s tests -p 'test_*.py'`
